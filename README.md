# SecrecyWeb
Esta é a implementação web do projeto Secrecy, um jogo RPG de turnos,com sistema completo de luta, desenvolvido em Javascript F(Vanilla JS)
O sistema opera como uma Single Page Application (SPA), gerenciando estados complexos de combate, inventário e narrativa sem a necessidade de recarregamento de página ou backends complexos. 
Toda a lógica é processada no Client-side, garantindo resposta imediata às ações do jogador.

Futuras Melhorias:
 Salvar progresso no navegador (localStorage).
 Responsividade aprimorada para dispositivos móveis.
 Expansão do banco de dados de inimigos, campanhas e cartas.
